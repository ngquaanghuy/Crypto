3. CAC THANH PHAN DA HOAN THANH
-------------------------------------------------------------------------------

  3.1. CLI COMMANDS

  Command      | Chuc nang                       | Algorithm mac dinh
  -------------+---------------------------------+-------------------
  encode       | Ma hoa dang text (string)       | base64
  decode       | Giai ma text                    | base64
  encrypt      | Ma hoa bang thuat toan          | aes-gcm
  decrypt      | Giai ma                         | aes-gcm
  protect      | Dong goi vao script Python      | aes-gcm
               | tu giai ma (self-decrypting)

  3.2. THUAT TOAN ENCODE (encode/decode)

  Ten      | Mo ta
  ---------+---------------------------------------------------
  base64   | A-Z a-z 0-9 +/, padding '='
  base32   | A-Z 2-7, padding '=', ho tro partial group
  base85   | Z85 alphabet, toi uu 'z' cho 4 bytes zero
  ascii85  | Adobe/PostScript, delimiter <~~>, 'z' shortcut
  hex      | Chu thuong 0-9 a-f, doc ca upper/lower
  xor      | XOR stream voi key tuy chinh

  3.3. THUAT TOAN ENCRYPT (encrypt/decrypt)

  Ten          | Che do          | Kiem tra tinh toan ven
  -------------+-----------------+-----------------------
  aes-ecb      | ECB             | HMAC-SHA256
  aes-cbc      | CBC             | HMAC-SHA256
  aes-ctr      | CTR             | HMAC-SHA256
  aes-gcm      | GCM (mac dinh)  | AEAD tag (built-in)
  chacha20     | Stream cipher   | HMAC-SHA256
  xor          | Stream XOR      | Khong

  Key derivation: PBKDF2-HMAC-SHA256, 100.000 iterations, salt 16 bytes

  3.4. OBFUSCATION (--obf <tech>)

  Technique  | Mo ta
  -----------+---------------------------------------------------
  rename     | Doi ten bien, ham, class -> ten ngan, kho doc
  strings    | Ma hoa string literal bang XOR, giai ma bang lambda
  vstrings   | Che string nang cao: cat string thanh nhieu doan,
             | moi doan XOR key khac nhau, giai ma bang eval long
  cleanup    | Xoa docstrings, string expressions doc lap
  flow       | Lam roi control flow co ban: if/else -> state machine
  aflow      | Lam roi control flow nang cao: state ID xao tron,
             | opaque predicates, dummy states, dispatch dict
  junk       | Chen code chet (dead code) voi opaque predicates
  mutate     | Bien doi bieu thuc: a+b -> lambda, x[y] -> .__getitem__
  all        | Ap dung TAT CA ky thuat

  3.5. ANTI-ANALYSIS (--anti-analysis <t>)

  Technique  | Mo ta
  -----------+---------------------------------------------------
  debug      | Phat hien debugger: sys.gettrace(), breakpointhook,
             | quet sys.modules phat hien pydevd/pdb/ipdb/...
  hook       | Phat hien hook: kiem tra __import__/compile/exec
             | co bi hook khong, phat hien sandbox
  scramble   | Trai cac doan anti-analysis vao nhieu vi tri khac
             | nhau trong luong giai ma, dung ten sys ngau nhien
  opaque     | Chen opaque predicates vao VM bytecode
  all        | Tat ca ky thuat

  3.6. COMPRESSION (--compress <algo>)

  Algo    | Thu vien
  --------+---------------------------
  none    | Khong nen
  zlib    | zlib (C++), mac dinh cho protect
  lzma    | Python's lzma (runtime)
  bz2     | Python's bz2 (runtime)
  brotli  | brotli (Python runtime)
  zstd    | zstandard (Python runtime)
  gzip    | gzip (Python runtime)
  lz4     | lz4.frame (Python runtime)
  snappy  | snappy (Python runtime)
  zopfli  | gzip module (Python runtime)
  blosc   | blosc (Python runtime)

  3.7. REGISTER VM (--vm)

  - Kien truc: 64 registers, bo nho constant, bang ten
  - Instruction: 8-byte fixed-size {op, rd, rs1, rs2, imm}
  - 33+ opcodes ho tro: load/store, so hoc, so sanh, call, jump,
    for loop, list/tuple, import, attribute access
  - Code-split pipeline: tach function/class defs (chay native)
    va module-level code (chay trong VM)
  - Ho tro ca Python 3.14 opcode specializations
  - IIFE (Immediately Invoked Function Expression) evaluation
    tai compile time
  - Opaque predicate injection
  - Tuong thich: Python 3.10+

  3.8. STUB GENERATION

  - Polymorphic: random names (30+), shuffle algorithm dispatch (11 blocks)
  - XOR key obfuscation voi random byte
  - Junk code injection
  - Ho tro compress (10 algos) + decompress tai runtime
  - Ho tro VM mode (version header 2)
  - Anti-analysis fragments gom 4 loai
